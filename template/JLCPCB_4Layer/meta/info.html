JLCPCB 4-Layer Template

Pre-configured project template optimized for JLCPCB 4-layer PCB fabrication.

Stackup

  • F.Cu - Signal (top)
  • In1.Cu - GND plane
  • In2.Cu - Power plane
  • B.Cu - Signal (bottom)

Total thickness: 1.6mm

Design Rules

  • Min trace width: 0.127mm (5mil)
  • Min trace spacing: 0.127mm (5mil)
  • Min via: 0.45mm / 0.2mm drill
  • Min hole: 0.2mm

Net Classes

  • Default - 0.2mm trace, 0.2mm clearance
  • Power - 0.4mm trace, 0.2mm clearance
  • High_Current - 0.8mm trace, 0.25mm clearance

Features

  • JLCPCB Gerber plot presets
  • Silkscreen text sized for legibility
  • Standard LCSC library integration
